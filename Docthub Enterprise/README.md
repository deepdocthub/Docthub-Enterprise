# Docthub Enterprise Scripts

This folder contains Docthub Enterprise automation and helper scripts organized by feature area.

## Structure

- `Candidate Search/` - Search and candidate-related automation scripts.
- `docthub_script/` - Shared Docthub automation utilities.
- `E Certificate/` - E-certificate generation and automation scripts.
- `Institute_Script/` - Scripts for institute/dashboard automation and testing.
- `Membership/` - Membership-related automation and data dump scripts.
- `Recruiter_scripts/` - Recruiter dashboard and form automation scripts.
- `test_example/` - Example test scripts.
- `Userapp_script/` - User application automation and inspection scripts.
- `X Path/` - XPath discovery and link automation tools.
- `Xpath/` - Additional XPath scripts.

## Usage

Open the relevant subfolder and run the Python script(s) for the area you want to use.

Example:
```powershell
python "Docthub Enterprise\Membership\manage membership 2.py"
```

## Notes

- These scripts are implemented in Python.
- Some scripts may depend on local data files or browser automation libraries.
- Review the specific script file for usage details and any required inputs.
